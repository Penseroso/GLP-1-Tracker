# GLP-1 Pipeline Board

Initial frontend skeleton for a GLP-1-focused biopharma pipeline intelligence web app.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Local typed mock data

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## MVP Scope

- Overview dashboard
- Searchable and filterable asset register
- Asset detail drawer
- Neutral placeholder records only

No scraping, authentication, backend, real database, alerts, or real company/product data are implemented in this MVP.
